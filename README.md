# Trim MP3 Cover Art

MP3ファイルに埋め込まれたカバー画像（アルバムアート）を中央基準で指定したアスペクト比（デフォルトは正方形）にトリミングし、再度埋め込むためのツールです。

元の画像がPNG形式である場合、JPEG形式に変換してから埋め込みます。

## 使用方法

MP3ファイルが含まれるディレクトリを指定して実行します。

```bash
trim_mp3_cover_art.sh <mp3ディレクトリのパス>
```

実行後、指定したディレクトリ内のすべてのMP3ファイルのカバーアートがトリミングされ、置き換えられます。

## 注意

- トリミングの結果は元ファイルに直接反映されるため、必要に応じて元のファイルをバックアップしてください。
